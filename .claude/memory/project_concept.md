Строим **Telegram-бот** (aiogram) как основной интерфейс. Это НЕ просто бэкенд-апи — TG-бот первичен.

**Флоу пользователя:**
1. Открывает бота в Telegram
2. Указывает виртуальную сумму депозита + период торгов (напр. "1000 USDC, 30 дней")
3. Бот запускает AI-анализ Polymarket и делает виртуальные ставки
4. React UI (Telegram Mini App / WebApp) показывает дашборд: позиции, P&L, поведение бота, сколько заработал
5. Деньги ВИРТУАЛЬНЫЕ — никакого реального кошелька, чистая симуляция

**UI на React** = Telegram Mini App (WebApp), открывается кнопкой внутри бота.

**Ключевое требование:**
- Каждый пользователь задаёт свой виртуальный капитал и период
- Бот показывает как себя вёл: сколько сделок, какой P&L, какая точность
- Симуляция — не бэктест на истории, а paper trading вперёд

**Why:** Доказать edge до риска реальными деньгами. Показать поведение бота пользователю наглядно.
**How to apply:** При любых изменениях архитектуры помнить: TG-бот = основной UI, React = Mini App внутри него, реальных денег нет.
